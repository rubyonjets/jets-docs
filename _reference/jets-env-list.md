## Usage

    jets env:list

## Description

List and show env vars


## Options

```
    [--format=FORMAT]                           # Output format: csv, dotenv, equal, info, json, markdown, space, tab, table
                                                # Default: dotenv
    [--reveal], [--no-reveal], [--skip-reveal]  # Reveal values also
                                                # Default: false
-n, [--function=FUNCTION]                       # Lambda Function name
                                                # Default: controller
```
