## Usage

    jets rollback VERSION

## Description

Rollback to a previous release

## Examples

    jets rollback 8

Use the [jets release:history](/reference/jets-release-history/) command to view release history.


## Options

```
-y, [--yes], [--no-yes], [--skip-yes]  # Skip are you sure prompt
```
