## Usage

    jets maintenance:on

## Description

Turn on maintenance mode


## Options

```
-y, [--yes], [--no-yes], [--skip-yes]  # Skip are you sure prompt
-r, [--role=ROLE]                      # Role to apply the maintenance mode to. IE: web worker
                                       # Default: web
```
