## Usage

    jets maintenance:status

## Description

Show maintenance mode status


## Options

```
-a, [--all], [--no-all], [--skip-all]  # Show status for all roles. Takes precedence over --role option
                                       # Default: false
-r, [--role=ROLE]                      # Role to apply the maintenance mode to. IE: web worker
                                       # Default: web
```
