Jets supports deploying your app as a Docker image.

## Advantages

* Increase app size allowance. Up to **10GB** images.
* A portable and more standard deployment format.

## Disadvantages

* Cold starts are slightly longer, a few milliseconds.
* You need to be little knowledgable about Docker.

Note: Jets automates the Docker build process and handles the heavy lifting for you.
